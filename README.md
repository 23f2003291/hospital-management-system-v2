# Hospital Management System V2

This project is developed for the course Modern Application Development II (MAD-II).

Roles in the system:
- Admin
- Doctor
- Patient

Features:
- Role-based authentication
- Admin doctor management
- Patient appointment booking
- Doctor treatment updates
- Appointment history
- Celery background jobs
- Redis caching

Tech Stack:
Flask
VueJS
SQLite
Redis
Celery
Bootstrap
